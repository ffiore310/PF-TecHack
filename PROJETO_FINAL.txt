================================================================================
                            RESUMO DO PROJETO
================================================================================

Sistema completo de varredura de vulnerabilidades web com interface grafica,
autenticacao multi-usuario e containerizacao Docker.

PRINCIPAIS FUNCIONALIDADES:
- Deteccao automatizada de vulnerabilidades (XSS, SQL Injection, CSRF)
- Sistema de autenticacao e autorizacao
- Dashboard interativo responsivo
- Execucao assincrona de scans
- Geracao de relatorios HTML detalhados
- Isolamento de dados por usuario
- Deploy containerizado

================================================================================
                         FASES IMPLEMENTADAS
================================================================================

FASE 1: Estrutura Basica
- Scanner de vulnerabilidades core
- Detectores especializados (XSS, SQLi, CSRF)
- Gerador de relatorios

FASE 2: Interface Web
- Dashboard Bootstrap 5
- Formularios de criacao de scans
- Visualizacao de resultados

FASE 3: Persistencia de Dados
- Banco de dados SQLite
- Modelos SQLAlchemy
- Historico completo

FASE 4: Sistema de Autenticacao
- Login/Logout/Registro
- Gestao de perfis de usuario
- Isolamento de dados
- Sessoes seguras
- Execucao assincrona

FASE 5: Containerizacao Docker
- Dockerfile otimizado
- Docker Compose
- Volumes persistentes
- Scripts de automacao
- Documentacao completa

                         COMO EXECUTAR
================================================================================

OPCAO 1 - COM DOCKER (Recomendado):
  ./docker-start.sh
  OU
  docker compose up -d

OPCAO 2 - SEM DOCKER:
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  python src/web/app_auth.py
